## ¿Qué es?

MiniAgent es tres cosas en un solo repositorio:

1. **Una IA publicitaria entrenable** — fork de minimind (26M parámetros), reentrenado con datos del dominio publicitario. El modelo aprende GAQL, estructura de campañas, estrategia de pujas, matemáticas PPC y análisis creativo.

2. **Ecosistema de servidores MCP de producción** — 14 conectores de plataformas publicitarias (Google, Meta, Microsoft, Amazon, Reddit, TradeDesk, LinkedIn y más), 80+ herramientas.

3. **Plataforma de habilidades para agentes** — habilidades para Claude Code, Codex, Gemini CLI para análisis publicitario, auditoría, operaciones seguras, cálculos PPC.

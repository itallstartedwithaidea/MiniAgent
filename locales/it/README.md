## Cos'è?

MiniAgent è tre cose in un unico repository:

1. **Un'IA pubblicitaria addestrabile** — fork di minimind (26M parametri), riaddestrato su dati del dominio pubblicitario. Il modello impara GAQL, struttura delle campagne, strategia di offerta, matematica PPC e analisi creativa.

2. **Ecosistema MCP server di produzione** — 14 connettori per piattaforme pubblicitarie (Google, Meta, Microsoft, Amazon, Reddit, TradeDesk, LinkedIn e altri), 80+ strumenti.

3. **Piattaforma di competenze per agenti** — competenze per Claude Code, Codex, Gemini CLI per analisi pubblicitaria, audit, operazioni sicure, calcoli PPC.

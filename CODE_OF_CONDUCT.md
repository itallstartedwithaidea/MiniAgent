# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

**Positive behavior**: constructive feedback, respectful disagreement, focusing on what's best for the community.

**Unacceptable behavior**: harassment, trolling, personal attacks, publishing private information.

## Enforcement

Report issues to john@itallstartedwithaidea.com. All reports will be reviewed and acted upon.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/).
